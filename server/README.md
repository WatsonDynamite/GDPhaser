## About

Server soft. for Grimdrive to run on

## How to use

```console
# Install dependencies
$ npm install

# Start the local development server (on port 8080)
$ ts-node --esm server.ts

# Ready for production?
# Build the production ready code to the /dist folder
$ npm run build
```
